AMQP Tutorials
==============

This is a port of the seven RabbitMQ tutorials to Serena. These have been adjusted in both wording
and code content to fit the idiomatic structurally concurrent style presented by this library.

These examples all assume you are using the `Trio`_ library as your main event loop runner.
